隐私政策

最后更新日期：2026年2月3日

重要提示：我们非常重视您的隐私保护。本应用采用本地存储方式，您的数据仅保存在您的设备上，我们不会收集或上传您的个人信息到服务器。

1. 信息收集

我们收集以下信息用于提供服务：

  • 账号信息：您注册时提供的手机号或邮箱地址
  • 游戏数据：您的游戏战绩、棋局记录
  • 第三方登录：如您使用Google/Facebook/Apple登录，我们会获取您授权的基本信息（昵称、头像）

2. 信息存储

本地存储：所有数据均存储在您设备的浏览器本地存储（localStorage）中，不会上传到任何服务器。

  • 密码经过 SHA-256 加密哈希后存储
  • 您可以随时清除浏览器数据来删除所有信息
  • 数据仅在您的设备上，其他人无法访问

3. 信息使用

收集的信息仅用于：

  • 身份验证和账号管理
  • 显示游戏统计和战绩
  • 改善游戏体验

4. 第三方服务

本应用集成以下第三方登录服务：

  • Google Sign-In：遵循 Google 隐私政策
  • Facebook Login：遵循 Facebook 数据政策
  • Sign in with Apple：遵循 Apple 隐私政策

使用第三方登录时，请查阅相应平台的隐私政策。

5. 数据安全

我们采取以下措施保护您的数据：

  • 密码使用 Web Crypto API 进行 SHA-256 哈希加密
  • 实施输入清理防止 XSS 攻击
  • 登录失败限制（5次失败后锁定15分钟）
  • 强制 HTTPS 加密传输（生产环境）
  • Content Security Policy (CSP) 安全策略

6. 儿童隐私

本应用适合所有年龄段用户。我们不会故意收集13岁以下儿童的个人信息。如果您是家长并发现您的孩子向我们提供了个人信息，请联系我们删除。

7. 您的权利

您拥有以下权利：

  • 访问权：查看您存储的所有数据
  • 删除权：通过清除浏览器数据删除所有信息
  • 修改权：随时修改您的账号信息
  • 注销权：随时注销账号

8. Cookie 使用

本应用不使用 Cookie，仅使用 localStorage 进行本地数据存储。

9. 隐私政策更新

我们可能会不时更新本隐私政策。更新后的政策将在应用中发布，并注明最后更新日期。建议您定期查看本政策。

10. 联系我们

如果您对本隐私政策有任何疑问，请通过以下方式联系我们：

  • 邮箱：support@example.com（请替换为真实邮箱）
  • GitHub Issues：提交反馈

11. 适用法律

本隐私政策受中华人民共和国法律管辖。对于欧盟用户，我们遵守 GDPR（通用数据保护条例）。

数据删除说明：

由于所有数据存储在本地，您可以通过以下方式完全删除数据：

  • 浏览器：设置 → 隐私 → 清除浏览数据 → 选择"Cookie和网站数据"
  • iOS Safari：设置 → Safari → 清除历史记录与网站数据
  • Android Chrome：设置 → 隐私设置 → 清除浏览数据
